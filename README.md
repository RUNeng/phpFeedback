# phpFeedback

一个轻量级的 PHP 开源反馈系统，致力于为个人项目或中小型网站提供简洁的反馈收集能力。

> 🌱 这是作者的第一个开源项目，还在持续成长中，欢迎提出建议！

## 特点

- **多项目支持**：可配置多个反馈项目，用户按项目分类提交
- **附件上传**：支持图片、视频附件（可配置开关和大小限制）
- **灵活联系方式**：支持电话、QQ、邮箱等多种联系方式的组合配置
- **匿名提交**：可开启匿名反馈功能
- **状态跟踪**：反馈状态支持「已提交」「进行中」「已采纳」「已修复」「未采纳」
- **本地缓存**：提交后自动缓存到本地，方便用户查看历史
- **编号查询**：用户凭编号或联系方式查询反馈进度
- **采纳公示**：已采纳/已修复的反馈单独展示，公开透明
- **深色模式**：支持手动切换亮色/暗色主题
- **响应式设计**：移动端和 PC 端均可良好适配

## 快速开始

### 使用说明

1. 访问 `/admin.html` 进入初始化页面，并设置管理员密码。
2. 登录后，如需重置或清除登录状态，请删除 `/data` 目录下的相关文件。

> ⚠️ 提示：`/data` 目录用于存储配置与缓存数据，删除其中的文件即可使系统恢复到初始化状态。
